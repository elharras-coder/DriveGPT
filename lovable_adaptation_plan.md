## 🎯 Порядок работы в Lovable

### Этап 1: Подготовка (15 мин)
1. Зайти в Lovable → проект Guest Assist Genius
2. **Duplicate** (создать копию)
3. Переименовать в «DriveGPT Bot Hosting»

### Этап 2: Убрать лишнее (промпт для Lovable)
```
Удали из бокового меню разделы: «Виджет на сайт», «Баланс и оплата», 
«Тарифы», «На сайт». Замени заголовок «Венера Резорт» на «DriveGPT» 
с иконкой робота 🤖
```

### Этап 3: Переделать «Отели» → «Проекты»
```
Переименуй раздел «Отели» в «Проекты». Каждая карточка проекта 
должна показывать:
- Название проекта
- Статус: зелёный бейдж «running» или серый «stopped»  
- Bot username: @bot_name
- Дата создания
- Кнопки: «Открыть», «Удалить»
Кнопка «Добавить отель» → «+ Новый проект»
```

### Этап 4: Переделать «Настройки ИИ» → «Python-код»
```
Замени раздел «Настройки ИИ» на «Python-код». Вместо текстовых полей 
для промптов сделай:
1. Зону drag-and-drop для загрузки ZIP-архива (подпись: «Перетащите 
   ZIP-архив или нажмите для выбора»)
2. Под ней — таблицу «Файлы проекта» с колонками: Имя файла, Размер
3. Кнопку «Проверить синтаксис» с результатом ✅ или ❌
4. Блок «Управление ботом» с кнопками «▶️ Запустить» и «⏹ Остановить»
5. Индикатор статуса: «🟢 Webhook активен» / «🔴 Остановлен»
6. Сворачиваемый блок «Логи» с моноширинным текстом
```

### Этап 5: Диалоги — мелкие правки
```
В разделе «Диалоги» замени надпись «AI-ассистент» на «Бот». 
Вместо «Гость» показывай имя пользователя из Telegram. Добавь 
иконку Telegram рядом с аватаром.
```

### Этап 6: Переделать «Пользователи» → «База данных»
```
Переименуй «Пользователи» в «База данных». Таблица должна содержать 
колонки: ID, Telegram ID, Username, Имя, Фамилия, Email, Генерации, 
Баланс, Первая покупка (✅/—), Дата. 
Добавь над таблицей кнопки: «📥 Экспорт CSV» и «📤 Импорт CSV».
При клике на «Импорт CSV» — модалка с зоной загрузки файла.
```

### Этап 7: Добавить «Логи заявок» (новый раздел)
```
Добавь новый раздел «Логи заявок» в боковое меню (иконка 📊). 
Содержимое:
- Над таблицей: два поля даты для фильтра (от — до)
- Таблица с колонками: ID, Пользователь, Username, Фото авто 
  (иконка 📷 со ссылкой), Фото диска (📷), Статус (бейдж), Дата
- Под таблицей: пагинация (кнопки с номерами страниц, по 25 строк)
```

### Этап 8: Подключение к API
```
Настрой все API-запросы на адрес https://api.drivegpt.pro. 
Используй fetch() для всех запросов. Базовый URL: /api/projects
```
